# Supabase setup voor recensies

Gebruik een tabel `reviews` met deze kolommen:

```sql
create table public.reviews (
  id bigint generated by default as identity primary key,
  name text not null,
  message text not null,
  rating int not null check (rating between 1 and 5),
  is_approved boolean not null default false,
  created_at timestamptz not null default now()
);
```

Zet daarna Row Level Security aan:

```sql
alter table public.reviews enable row level security;
```

Policy om goedgekeurde reviews publiek leesbaar te maken:

```sql
create policy "Public can read approved reviews"
on public.reviews
for select
using (is_approved = true);
```

Policy om nieuwe reviews vanaf de website toe te voegen:

```sql
create policy "Public can create reviews"
on public.reviews
for insert
with check (is_approved = false);
```

Vul daarna in [script.js](script.js) deze waarden in:

- `SUPABASE_URL`
- `SUPABASE_ANON_KEY`

Belangrijk:

- Gebruik alleen de anon/public key in de browser.
- Laat reviews eerst op `is_approved = false` binnenkomen.
- Keur reviews handmatig goed in Supabase voordat ze zichtbaar worden.

Voor moderatie kun je in Supabase Table Editor per review `is_approved` op `true` zetten.
